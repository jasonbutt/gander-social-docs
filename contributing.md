# How to Contribute

Thanks for helping improve the Gander Social user guide.
You do **not** need to be technical to contribute.

## Option 1: Report an issue (easiest)
If something is wrong, outdated, or confusing:

1. Click the **Issues** tab
2. Click **New Issue**
3. Describe what’s wrong or missing

That’s it.

## Option 2: Suggest an edit
If you know what the fix should be:

1. Open the page you want to change
2. Click the ✏️ Edit button
3. Make your change
4. Submit it as a suggestion

A maintainer will review it before publishing.

## Guidelines
- Be respectful
- Focus on how the product works today
- Avoid speculation or roadmap promises

Thanks for helping keep this guide useful.
